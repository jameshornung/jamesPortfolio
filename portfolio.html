Run This Townclose

This is a website built to combine two of my passions: running and web development. It is designed to combine all the resources you need to enjoy running in the Austin community in one place. There are route maps and descriptions, training plans, strength training workouts, training videos, a race calendar, and a community bulletin board. This was built using a materialize UI framework, JavaScript, jQuery, Node, Express, Handlebars, and MongoDB. It features a user login, using the passport local strategy, and a fully responsive design that allows you to take it with you on a run.

Hangman: ACL Fest 2016close

A JavaScript powered hangman game featuring artists from the 2016 ACL Festival lineup.

Key highlights from this game include:

  • • Selects artists at random and does not repeat any artist during an individual game session.

  • • Accepts only alphabetic characters as user inputs.

  • • Alerts the user if an input is invalid, or has already been guessed.

  • • Tracks overall wins and losses.

  • • Get it right, or run out of guesses, and the game will provide a picture and short bio for the selected artist, then automatically moves on to the next challenge.
